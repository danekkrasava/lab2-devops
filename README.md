# Лабораторная работа №2: Изучение базовых команд Linux

## Цель работы:Освоить базовые команды Linux для навигации по файловой системе, управления файлами и каталогами, а также изучить разницу между жёсткой и символической ссылками.


## 1. Навигация по файловой системе
Была изучена команда pwd (print working directory), которая показывает текущий каталог. Затем с помощью команды cd / выполнен переход в корневой каталог. Команда ls использована для просмотра содержимого корневого каталога.(см. рис.1)
2. Создание каталогов и файлов
С помощью команды mkdir создан каталог test, внутри него — каталог test2. Командой touch создан пустой файл text. Командой mv файл переименован в textSIT. Командой cp создана копия файла с именем copy.txt. Командой ln создана жёсткая ссылка link на файл copy.txt. Командой ln -s создана символическая ссылка simlink на тот же файл. Результат просмотрен командой ls -la. (см рис. 2,3,4)
3. Удаление файлов и каталогов
Командой rm удалены все созданные файлы и ссылки. Затем выполнен выход в домашний каталог и командой rm -rf удалён каталог test. (см. рис. 5)
